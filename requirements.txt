# ============================================================
#  INSTALLATION NOTES FOR WINDOWS + VS CODE
#
#  1. Create virtual environment:
#     python -m venv venv
#     venv\Scripts\activate
#
#  2. Upgrade pip first:
#     python -m pip install --upgrade pip
#
#  3. Install PyTorch CPU (paste this FIRST before pip install):
#     pip install torch==2.4.0 --index-url https://download.pytorch.org/whl/cpu
#
#  4. Then install everything else:
#     pip install -r requirements.txt
#
#  5. Install Playwright browsers (one-time):
#     playwright install chromium
#
#  6. If unstructured[pdf] fails, install poppler:
#     Download from: https://github.com/oschwartz10612/poppler-windows/releases
#     Extract and add bin/ folder to your Windows PATH
#
# ============================================================
